# 🚀 QUICK START GUIDE - Employee Management System

## 📦 FOR COMPANIES: SIMPLEST DEPLOYMENT

### 🎯 ONE-CLICK INSTALLATION
1. **Download** the system files to the server computer
2. **Right-click** `INSTALL_SERVICE.bat` 
3. **Select** "Run as Administrator"
4. **Wait** for "Installation Complete" message
5. **Done!** System runs automatically forever

### 🌐 INSTANT ACCESS
- **Local**: `http://localhost:8080`
- **Network**: `http://[SERVER-IP]:8080`
- **Login**: `admin@company.com` / `admin123`
